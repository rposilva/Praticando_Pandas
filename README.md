#  Porta de entrada para Pandas [![official JetBrains project](http://jb.gg/badges/official.svg)](https://confluence.jetbrains.com/display/ALL/JetBrains+on+GitHub)

Bem-vindo ao curso prático da JetBrains Educational Products, dedicado a dominar uma das bibliotecas Python mais populares — Pandas. O nome "Pandas" tem suas raízes no termo "dados em painel", que é um termo econométrico para conjuntos de dados.

Este curso foi criado para indivíduos que possuem conhecimento básico de programação Python. Tem como objetivo fornecer uma cobertura abrangente dos seguintes objetivos:

1. **Compreendendo as estruturas de dados do Pandas:** aprofunde-se nas complexidades do DataFrame e da Série. 
2. **Dominar a preparação e limpeza de dados:** resumir, filtrar, classificar.
3. **Realização de exploração, manipulação e análise de dados** — um passo fundamental em diversos domínios, desde competições de ML até tarefas de negócios.
4. **Abordando tópicos avançados:** Interaja com dados tabulares e implemente seu próprio modelo de recomendação do TED Talks para aprofundar seu conhecimento e proficiência com o Pandas.
5. **Fornecendo vários exemplos reais de implementação de código** e exercícios práticos para testar suas novas habilidades com feedback instantâneo.

O curso aproveita a documentação oficial do Pandas, complementada com exemplos e exercícios adicionais inspirados em cenários do mundo real.

Ao navegar neste curso, lembre-se de que o objetivo é aprender e se divertir. Boa sorte!

Caso deseje contribuir, sinta-se à vontade para enviar uma solicitação de pull para o curso [Repositório GitHub](https://github.com/jetbrains-academy/gateway-to-pandas).
